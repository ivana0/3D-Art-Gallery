> User controls: 
	- display/hide with 'h' key 
	- 4, 5, 6, 7 keys not implemented

	H - User Interaction Help Menu
	ESC - quit program
	---------	CAMERA	----------
	"W/S - pitch up/down
	"A/D - yaw left/right
	"PAGE UP/PAGE DOWN - move up/down
	"UP/DOWN/LEFT/RIGHT - move forward/backward/left/right
	"HOME - return to initial position
	
	------WIREFRAME MODE-------
	1 - toggle white wireframe with black background
	2 - toggle black wireframe with white background
	3 - toggle yellow wireframe with blue background

	--------SOLID MODE--------- //---> not implemented
	4 - colored without lighting or textures
	5 - shaded using lighting
	6 - textured without lighting
	7 - shaded using lighting and textured


	----------	LIGHTS	-----------
	F1 - left blue light
	F2 - right red light
	F3 - middle globe light
	F4 - back globe light
	F5 - spotlight 1 inside of the gallery
	F6 - spotlight 2 in front of the gallery
	'+' - day mode
	'-' - night mode
	L/l - toggle blinking lights on/off
